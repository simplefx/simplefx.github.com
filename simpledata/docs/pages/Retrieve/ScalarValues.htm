Simple.Data

Simple.Data defines a number of commands for retrieving data from a data store. These can then be daisychained in a LINQ-like fashion with further methods to modify the basic query. For example, Simple.Data includes four similar methods to cast the result of a query back into its simple type or a collection of scalar types instead of having to retrieve it as the property of a SimpleRecord with an empty string.

ToScalar

ToScalar() returns an object.

ToScalar<T> returns a scalar value of simple type T.

As long as your query returns one row with one column, it will return that single value. If there is not exactly one row, an exception will be thrown

ToScalarList

ToScalarList() returns a List of objects.

ToScalarList<T> returns a List of type T.

ToScalarArray

ToScalarArray() returns an array of objects.

ToScalarArray<T> returns an array of type T.

ToScalarOrDefault

ToScalarOrDefault() returns a single value or null if no row is found.

ToScalarOrDefault<T> returns a value of type T or default<T> if no row is found.

If more than one row is returned, an exception will be thrown
