Simple.Data

Simple.Data defines a number of commands for retrieving data from a data store. These can then be daisychained in a LINQ-like fashion with further methods to modify the basic query.

GetCount

Use the GetCount method to return an integer count of null and non-null data fields in a table given a set of criteria.

Examples

1

Run GetCount() on a Table reference with no criteria

db.Albums.GetCount();

Results in the following SQL

select count(*) from albums

2

Run GetCount() on a Table reference with a simple criteria

db.Albums.GetCount(db.Albums.GenreId != null);

Results in the following SQL

select COUNT(*) from [dbo].[Albums] WHERE [dbo].[Albums].[GenreId] IS NOT NULL

3

Run GetCount() on a Table reference with complex criteria

db.Albums.GetCount(db.Albums.GenreId == 2 || db.Albums.ArtistId == 160);

Results in the following SQL

select COUNT(*) from [dbo].[Albums] WHERE ([dbo].[Albums].[GenreId] = @p1 OR [dbo].[Albums].[ArtistId] = @p2)
@p1 (Int32) = 2
@p2 (Int32) = 160
