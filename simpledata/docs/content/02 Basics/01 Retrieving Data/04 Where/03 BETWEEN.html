BETWEEN

It is possible to use Simple.Data to perform Range and Array operations. In SQL terminology, a Range uses the BETWEEN clause and an Array uses the IN (n..n1..n2) clause. These work work with the FindAll methods described elsewhere.

FindAll with Integer Range

Simple.Data:

            _db.Users.FindAllById(1.to(10));

Generated SQL:

  @p1_start = 1
  @p1_end = 10
  select [dbo].[users].[id],[dbo].[users].[name],[dbo].[users].[password],[dbo].[users].[age],[dbo].[users].[joindate] 
  from [dbo].[users] where [dbo].[users].[id] between @p1_start and @p1_end

FindAll with DateTime Range

Simple.Data:

            _db.Users.FindAllByJoinDate("01/01/2011".to("31/01/2011"))

Generated SQL:

@p1_start (Object) = 01/01/2011 00:00:00
@p1_end (Object) = 31/01/2011 00:00:00

select [dbo].[Users].[Id],[dbo].[Users].[Name],[dbo].[Users].[Password],[dbo].[Users].[Age],[dbo].[Users].[JoinDate] from [dbo].[Users] 
WHERE [dbo].[Users].[JoinDate] BETWEEN @p1_start AND @p1_end
