Retrieving Data

There are six Simple.Data commands you can use to return data from a data store.

Name Returns Description
All SimpleQuery The All method returns all data from a table.
FindBy* SimpleRecord The FindBy methods return a single row of data; if more than one row matches, the first row retrieved from the data store is returned.
FindAllBy* SimpleQuery The FindAllBy methods are for finding multiple rows of data.
Get SimpleRecord The Get method returns data from a table where the table’s primary key matches its argument values. The Get method returns a single row of data; if more than one row matches (not that it should unless your primary key integrity is broken), the first row retrieved from the data store is returned.
Find SimpleRecord The Find method returns a single row of data; if more than one row matches, the first row retrieved from the data store is returned.
FindAll SimpleQuery The FindAll method returns multiple rows of data.

Notes

  • The FindBy and FindAllBy methods are used to return data from a table based on a set of criteria where the value of columns in a row are equal to given values - e.g. ArtistId=6 and Title="Back in Black" - or within certain sets of values.
  • The Find and FindAll methods are used when the criteria is more complicated than simple equality tests. E.g. Title LIKE "Back%". Criteria can be constructed just using regular C# expressions, with all operators supported and precedence-grouping honoured.
  • Still included in Simple.Data for legacy reasons, the Query and QueryBy methods are now deprecated in favour of using All and FindAllBy respectively.
