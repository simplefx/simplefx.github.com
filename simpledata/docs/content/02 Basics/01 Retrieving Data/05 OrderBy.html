Sorting Data

Once you have run a Finding Data query, you can then sort the returned data at the Sql Level by using the Order By clause. This clause comes in two forms, OrderBy with the column name as part of the method call, or with an expression supplied to the method.

Order By (Method name form)

Simple.Data:

            _db.Customers.FindAll(_db.Customers.CustomerName == "Arthur")
                .OrderByCustomerName();

Generated SQL:

  @p1 = 'Arthur'
  select [dbo].[customers].[customerid],[dbo].[customers].[customer_name] 
  from [dbo].[customers] where [dbo].[customers].[customer_name] = @p1 order by [customer_name]

Order By (Expression form)

Simple.Data:

            _db.Customers.FindAll(_db.Customers.CustomerName == "Arthur").OrderBy(_db.Customers.CustomerName)

Generated SQL:

  @p1 = 'Arthur'
  select [dbo].[customers].[customerid],[dbo].[customers].[customer_name] 
  from [dbo].[customers] where [dbo].[customers].[customer_name] = @p1 order by [customer_name]
