FindBy

The Find and FindAll methods are used when the criteria is more complicated than simple equality tests. Criteria can be constructed just using regular C# expressions, with all operators supported and precedence-grouping honoured.

Find/FindAll Expressions

There are many functions available for you to use within the generated WHERE Clause. This is not an exhaustive list, but demonstrates a number of options.

Arithmetic Operators

Sql Examples (MSDN)

+ (Add)

Simple.Data:

  var user = db.Users.Find(_db.Users.Id + _db.Users.Age == 42);

Generated SQL:

  @p0 = 42
  select [dbo].[Users].* from [dbo].[Users] where [dbo].[Users].[Id] + [dbo].[Users].[Age] = @p0
