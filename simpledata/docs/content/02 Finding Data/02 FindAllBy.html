FindAllBy

The FindAllBy methods are for finding multiple rows of data.

The return value from a FindAllBy method is a SimpleQuery; you may start enumerating this directly, which will execute the query at the back-end, or you can append further Query methods for operations like sorting, paging etc. See Querying for more info.

There are two ways to form a FindAllBy method: you may specify the column names as part of the method, or as named parameters. You can use whichever you prefer, unless you run into an edge-case which forces the named parameter form (more on that later).

Method name form

Simple.Data:

  var user = db.Users.FindAllByType("Premium");

Generated SQL:

  @p0 = 'Premium'
  SELECT * FROM Users WHERE Type = @p0

Named parameter form

Simple.Data:

  var user = db.Users.FindAllBy(Type:"Premium");

Generated SQL:

  @p0 = 'Premium'
  SELECT * FROM Users WHERE Type = @p0
