FindBy

The FindBy methods return a single row of data; if more than one row matches, the first row retrieved from the data store is returned.

There are two ways to form a FindBy method: you may specify the column names as part of the method, or as named parameters. You can use whichever you prefer, unless you run into an edge-case which forces the named parameter form (more on that later).

Method name form

Simple.Data:

  var db = Database.Open();
  var user = db.Users.FindByUserName('bob');
  Console.WriteLine(user.FullName);

Generated SQL:

  @p0 = 'bob'
  SELECT * FROM Users WHERE UserName = @p0
